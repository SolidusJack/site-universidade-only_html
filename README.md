# Site Universidade – HTML

Projeto de estudo desenvolvido utilizando apenas HTML.

O objetivo deste projeto foi praticar a estruturação de páginas web, organização de conteúdo e uso de tabelas para layout, simulando um site institucional de uma universidade.

O site possui páginas como:
- Página inicial
- Sobre a universidade
- Página de contato

Este projeto faz parte do meu processo de aprendizado em desenvolvimento web.
